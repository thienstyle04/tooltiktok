# Dalat carousel tool

Frontend hien da chuyen sang Next.js.

## Chay local

1. Cai dependencies:
   - `cd backend && npm install`
   - `cd frontend && npm install`
2. Chay ca backend va frontend:
   - `npm run dev`

URL:
- Next.js frontend mac dinh: `http://127.0.0.1:3001/`
- NestJS API mac dinh: `http://127.0.0.1:3000/`

Neu port mac dinh dang ban, `npm run dev` tu dong chon port tiep theo va in URL thuc te.

Frontend Next.js proxy `/api/*` va `/assets/*` ve backend.
